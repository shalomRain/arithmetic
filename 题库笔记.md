###### 2021.03.04

354 俄罗斯套娃

如果有多维排序,可以直接用sort方式

```javaScript
let envelopes = [ [ 46, 89 ], [ 52, 68 ], [ 50, 53 ], [ 72, 45 ], [ 77, 81 ] ]

envelopes.sort((e1, e2) => {
  // 如果前面的一个数值不等于后面一个值,则以前面的值按从小到大排序(注意这里的else在参考答案基础上做了修改)
  if (e1[0] !== e2[0]) {
    return e1[0] - e2[0]
  } else {
    return e1[1] - e2[1]
  }
})
```
